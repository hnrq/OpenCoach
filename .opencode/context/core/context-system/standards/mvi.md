# MVI Principle (Minimal Viable Information)

**Purpose**: Extract only core concepts, not verbose explanations.

---

## The Formula

```
Core Concept (1-3 sentences)
  ↓
Key Points (3-5 bullets)
  ↓
Quick Example (Coaching data or Code)
  ↓
Reference Link (full docs)
```

## Why it matters for OpenCoach
Coaching generates long logs. MVI ensures the Head Coach only sees the **metabolic truth** and **current strategy**, not the chat history.
