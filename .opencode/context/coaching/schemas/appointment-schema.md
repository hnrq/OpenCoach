# Appointment File Schema

| Property                         | Description                                                                                                                  | Type                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| :------------------------------- | :--------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`date`** (\*)                  | Appointment date in YYYY-MM-DD format                                                                                        | `string` (_regex: `/^\d{4}-\d{2}-\d{2}$/`_)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |

| **`decisions`** (\*)             | Each methodology override or constraint raised during the appointment. | **`preferences_delta`** (\*)     |                                                                                                                              | _Object with properties:_<ul><li><b><code>food_swaps</code></b> (\*): <i>Array of objects:</i><ul><li><b><code>remove</code></b> (\*): <code>string</code> - Food being replaced</li><li><b><code>add</code></b> (\*): <code>string</code> - Food replacing it</li></ul> - Food substitutions raised during the appointment. Leave [] if none.</li><li><b><code>food_additions</code></b> (\*): <code>Array<string></code> - New foods added to the athlete's primary list. Leave [] if none.</li><li><b><code>food_removals</code></b> (\*): <code>Array<string></code> - Foods dropped from the athlete's plan. Leave [] if none.</li><li><b><code>constraints_added</code></b> (\*): <code>Array<string></code> - New injury or equipment constraints, e.g. 'no overhead pressing'. Leave [] if none.</li><li><b><code>constraints_removed</code></b> (\*): <code>Array<string></code> - Constraints that no longer apply, e.g. 'knee pain resolved'. Leave [] if none.</li><li><b><code>schedule_changes</code></b> (\*): <i>Array of objects:</i><ul><li><b><code>field</code></b> (\*): <code>string</code> - Schedule field being changed, e.g. 'rest_days', 'sport_days'</li><li><b><code>value</code></b> (\*): <code>unknown</code> - New value, e.g. ['sunday'] or 'tuesday'</li></ul> - Schedule shifts raised during the appointment. Leave [] if none.</li></ul> |

'Legs moved to Saturday for sport recovery window'  | `Array<string>`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
